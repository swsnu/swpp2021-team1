## Start FrontEnd
- `cd frontend`
- `yarn install`
- `yarn start`

## Start BackEnd
- `cd backend`
- `cd rejourn`
- `python3 manage.py makemigrations project`
- `python3 manage.py migrate`
- `python3 manage.py runserver`
